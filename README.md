# Survival Voxel

Three.js を使用して構築された、サバイバル要素を持つボクセルゲームエンジン（Minecraft クローン）です。

## 🚀 主な機能

- **ボクセルエンジン**: チャンクベースの描画システムと Web Worker を使用したバックグラウンド処理により、安定した動作を実現。
- **地形生成**: パーリンノイズ（Simplex Noise）を使用した無限に広がる地形生成。
- **サバイバルシステム**:
  - **Health & Hunger**: ダメージや空腹の管理。食べ物を食べて回復する仕組み。
  - **昼夜サイクル**: 動的なライティングと時間の経過。
- **インベントリ & クラフト**:
  - アイテムの収集と管理。
  - クラフトシステム、精錬（かまど）システムの搭載。
  - ホットバーによるアイテムの使用。
- **インタラクション**:
  - ブロックの設置と破壊。
  - ドロップアイテムの回収。
  - 足音や環境音などのサウンドシステム。
- **高度な UI**:
  - ミニマップ。
  - インベントリ、クラフト、ポーズ画面。
  - ステータスバー（HP、空腹、時間）。
- **セーブ/ロード**: プレイヤーの位置、所持品、変更されたブロックの状態を保存。

## 🛠 技術スタック

- **Core**: [Three.js](https://threejs.org/) (WebGL)
- **Framework**: Javascript (ES Modules)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Style**: CSS (Vanilla CSS)

## 📦 セットアップ

### 必要条件

- [Node.js](https://nodejs.org/) (LTS 推奨)

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

### ビルド

```bash
npm run build
```

## 📂 プロジェクト構造

- `src/core`: ゲームエンジン、イベントバス等のコアロジック
- `src/world`: チャンク、地形生成、ブロックデータ
- `src/player`: コントローラー、物理演算、ステータス（HP/空腹）
- `src/rendering`: テクスチャ、マテリアル、パーティクル
- `src/inventory`: インベントリ、クラフト、精錬システム
- `src/ui`: 各種 UI コンポーネント
- `src/interaction`: レイキャスト、ブロックアクション、アイテムドロップ
- `src/storage`: セーブデータ管理
- `styles`: CSS デザイン
